# Frontend

React + Vite application. All client-side code lives here.

Scaffold this folder with Vite when starting a new project:

```bash
npm create vite@latest frontend -- --template react-ts
cd frontend && npm install
```

## Running locally

```bash
cd frontend
npm install
npm run dev
```

Runs at `http://localhost:5173` by default.

## Build for production

```bash
cd frontend
npm run build
```

Output goes to `frontend/dist/`.
