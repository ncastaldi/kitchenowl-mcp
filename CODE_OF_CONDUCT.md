# Code of Conduct

## Our pledge

This project is maintained as a professional, welcoming space. Everyone who interacts with this repository — through issues, pull requests, or discussions — is expected to be respectful and constructive.

## Expected behavior

- Be respectful and considerate in all communication
- Accept constructive feedback gracefully
- Focus on what is best for the project

## Unacceptable behavior

- Harassment, discrimination, or personal attacks of any kind
- Dismissive or unconstructive criticism
- Any conduct that would be considered inappropriate in a professional setting

## Enforcement

This project is maintained by a single developer. Violations can be reported by opening a private security advisory or emailing the maintainer directly (see profile). Reports will be reviewed and addressed at the maintainer's discretion.

## Scope

This code of conduct applies to all project spaces including the repository, issues, pull requests, and any other communication channels associated with the project.
